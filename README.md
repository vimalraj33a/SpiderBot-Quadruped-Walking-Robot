# SpiderBot-Quadruped-Walking-Robot
SpiderBot is a 4-legged quadruped walking robot built with a 3D-printed body, 8 servo motors, and an ESP32 microcontroller running custom firmware. It uses ROS2 Humble as the middleware, with Gazebo Classic for physics simulation and MoveIt2 for motion planning across 12 named poses including stand, walk, wave, and stretch.
